A high-end, cinematic front-end web application paying tribute to the legendary anime, *Attack on Titan (Shingeki no Kyojin)*. This project is designed to deliver a visually striking, interactive storytelling experience using modern web technologies. 

It serves as a showcase of advanced front-end development skills, featuring dynamic scroll-based animations, premium UI/UX design, and custom art generation.

### 🌐 [View Live Demo Here](https://your-deployed-link.vercel.app/)

---

## ✨ Features

- **Cinematic Hero Section:** A highly engaging video background with synchronized parallax text reveals.
- **Scroll-Triggered Animations:** Heavy use of GSAP's `ScrollTrigger` plugin for immersive reveals, sticky sections, and horizontal scrolling arrays.
- **Responsive Layouts:** Carefully crafted CSS utilizing `clamp()`, `flexbox`, and CSS variables to ensure the experience is flawless on desktop, tablet, and mobile.
- **Glassmorphism UI:** Translucent cards with deep backdrop blurs, rich box shadows, and hover micro-animations to create a premium aesthetic.
- **Custom Visuals:** Incorporates custom-generated, dark anime-style portraits and landscapes to fully capture the mood of the series.
- **Dynamic Timeline:** A vertical SVG-styled timeline that "bleeds" down the page as the user scrolls, chronicling the history of the walls.

## 🛠️ Technology Stack

- **Framework:** React 18
- **Build Tool:** Vite
- **Styling:** Vanilla CSS (Custom Properties, Grid/Flexbox, BEM-inspired architecture)
- **Animation:** GSAP (GreenSock Animation Platform) + ScrollTrigger
- **Typography:** Google Fonts (Cinzel, Cinzel Decorative, IM Fell English)

## 🚀 Getting Started

If you want to run this project locally and explore the code:

### Prerequisites
- Node.js (v16 or higher)
- npm or yarn

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/your-username/attack-on-titan-experience.git
   ```

2. **Navigate to the project directory:**
   ```bash
   cd attack-on-titan-experience
   ```

3. **Install dependencies:**
   ```bash
   npm install
   ```

4. **Start the development server:**
   ```bash
   npm run dev
   ```

5. **View the app:**
   Open `http://localhost:5173` (or the port provided in your terminal) in your browser.

## 🎨 Design Philosophy

The aesthetic of this project is deeply rooted in the tone of the source material: **dark, tragic, yet undeniably beautiful.** 
Instead of a standard informational webpage, the goal was to build an *experience*. 
- **Color Palette:** Dominated by a deep abyss/void `#0a0a0f`, accented with striking blood red `#b82020` and muted regal gold `#c4a450`.
- **Typography:** Utilizing serif fonts like `Cinzel` and `IM Fell English` to evoke an ancient, classical, and historical manuscript feel.
- **Motion:** Every animation is deliberately tuned. Slow pans and scale changes emphasize scale and dread, while rapid fades and horizontal scrolls mimic the fast-paced action of the Survey Corps.
